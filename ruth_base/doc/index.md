# Ruth Base

Ruth Base collects 3rd party JavaScript and style libraries used in web coding, which doesn't have any Rails gem.

## Installation

Add this line to your application's Gemfile:
```ruby
gem "rug_base"
```

And then execute:

```bash
$ bundle install
```
