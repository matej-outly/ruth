## Installation

Font Awesome is available as gem for Rails.

Add into your Gemfile and run `bundle install` afterwards:
```ruby
# Font awesome
gem "font-awesome-sass", "~> 4.6.2"
```

Add style into `application.scss` file:
```scss
@import "font-awesome-sprockets";
@import "font-awesome";
```
